Chapter 2: Variables and Simple Data Types

This chapter introduces the concept of variables and some basic data types in Python such as strings, numbers, and comments. Additionally, we'll explore the Zen of Python, which encapsulates Python's design principles.

Variables

Variables store data values in Python. Assigning a value to a variable in Python is simple:

Example: 
my_variable = 10

In this example, my_variable is a variable that stores the value 10.

Strings

Strings are sequences of characters enclosed in single or double quotes. Here's how you can work with strings in Python:

Example: 
greeting = "Hello, World!"

This code assigns the string "Hello, World!" to the variable greeting.

Numbers

Python supports different types of numbers including integers, floating-point numbers, and complex numbers. You can perform basic arithmetic operations with them:

Example: 
result = 5 + 3 * 2

In this example, the result of the arithmetic expression will be 11.

Comments

Comments are used to explain your code and make it more readable. They are ignored by the Python interpreter and can be created using a hash symbol (#):

Example: 
# This is a comment

Comments help clarify your code without affecting how it runs.

The Zen of Python

The Zen of Python is a collection of aphorisms that capture Python’s philosophy. You can display these principles by running the following in the Python interpreter:

Example: 
import this

This will display a list of guiding principles that inform Python’s design, such as "Beautiful is better than ugly" and "Explicit is better than implicit."
