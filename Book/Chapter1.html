Chapter 1: Getting Started

In this chapter, we will walk through the basics of setting up Python, running your first program, and understanding how to work across different operating systems.

Setting up Your Programming Environment

Before starting to code in Python, you need to set up your programming environment. This involves downloading Python, installing a code editor, and making sure your system is ready to run Python scripts.

Info: You can download Python from the official website: Python Downloads.
Steps:
  1. Download Python for your system.
  2. Install a code editor like VS Code or PyCharm.
  3. Ensure Python is added to your system's PATH.

Python on Different Operating Systems

Python works on a variety of operating systems including Windows, macOS, and Linux. Each system has slightly different ways of installation and usage, but the core principles remain the same.

Tip: macOS and most Linux distributions come with Python pre-installed. However, it's often recommended to install the latest version.

Running Python Programs from a Terminal

You can run Python programs directly from your terminal or command line. Open a terminal, navigate to the folder containing your Python file, and type:

Command: 
python hello_world.py

This will execute the Python script and display the output in the terminal.

In this chapter, we covered the basics of setting up Python, installing a code editor, running a simple Python program, and working across different operating systems. You should now be able to run your first Python program confidently!
